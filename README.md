# Dashboard de Trocas – Power BI

## 📊 Visão Geral
Este projeto apresenta um dashboard interativo em Power BI para análise de trocas com:

- Estruturação inicial do painel.
- Narrativa automática manual (via medidas DAX).
- Inserção de slicers/filtros interativos.
- Criação de KPIs executivos em cartões visuais.

## 🧩 Funcionalidades implementadas

- KPIs principais: Total de Trocas, Valor Total, Ticket Médio.

## ⚙️ Tecnologias utilizadas
- Power BI Desktop (versão 2020+).
- DAX para criação de medidas dinâmicas.
- GitHub para versionamento e portfólio.
